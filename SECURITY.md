# Security Policy

## Supported versions

Only the latest ScenePulseN release is supported. ScenePulseN requires
SillyTavern 1.18.0 or newer.

## Reporting a vulnerability

Do not publish exploit details in a public issue.

Use GitHub's private vulnerability reporting feature for this repository when
available. Otherwise contact the ScenePulseN maintainer through the
[l1kiru GitHub profile](https://github.com/l1kiru) and provide:

- affected ScenePulseN and SillyTavern versions;
- reproduction steps;
- expected and observed behavior;
- impact assessment;
- a minimal proof of concept, without private chat content or API keys.

Upstream ScenePulse vulnerabilities that are not specific to ScenePulseN should
also be reported to the upstream project.
